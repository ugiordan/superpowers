# Layer 2: Visual Overview Template

## Purpose
2-minute visual understanding — architecture diagram and key metrics.

**Scope:** This layer owns visuals. Do NOT include approach comparison (Layer 3) or code walkthrough (Layer 4).

## Format

```markdown
## Layer 2: Visual Overview

### Architecture

[Mermaid diagram showing the system structure after this change]

### Key Metrics

| Metric | Before | After | Change | Target |
|--------|--------|-------|--------|--------|
| [Metric 1] | [Value] | [Value] | [Delta] | [Goal] |
| [Metric 2] | [Value] | [Value] | [Delta] | [Goal] |

### Component Relationships

[If multiple components interact, show a sequence or flow diagram]
```

## Diagram Selection

See `utils/visualization-helper.md` for guidance on choosing between Mermaid, tables, and ASCII diagrams.
